# Web Scraping using BeautifulSoup
## Requirements:
> Python 3, BeautifulSoup, Requests
## Installations:
### Python 3:
```
https://www.python.org/downloads/
```
### BeutifulSoup
```
pip install bs4
```
### Requests
```
pip install requests
```
### How to Run
> Windows
```
python main.py
```
> Linux
```
python3 main.py
```
